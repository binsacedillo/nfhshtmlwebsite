C++

A general-purpose programming language that is used for creating high-performance applications, such as operating systems, game engines, and scientific simulations. It is an extension of the C programming language and allows for both low-level memory manipulation and high-level abstraction.

Python

A high-level, interpreted programming lanuage that emphasizes code readability and simplicity. It is often used for scripting, data analysis, and machine learning.

Java

A popular programming language that is used to create a wide range of applications, from Android apps to enterprise-level software. It is known for its write-one-run-anywhere approach, meaning that code written for one platform can be easily adapted to run on another.
